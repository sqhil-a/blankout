# Blankout Privacy Site

Static privacy policy website for Blankout.

## Files

- `index.html` - primary privacy policy page
- `privacy.html` - duplicate App Store-friendly privacy URL
- `assets/styles.css` - site styles
- `.nojekyll` - keeps GitHub Pages serving static files directly

## GitHub Pages

After pushing, enable Pages in GitHub:

1. Open the repository settings.
2. Go to Pages.
3. Set Source to `Deploy from a branch`.
4. Select `main` and `/root`.
5. Save.

The privacy URL should become:

`https://sqhil-a.github.io/blankout/`
